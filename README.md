# mitzi-tyaid

<img alt="Main Screen"  src="screenshots/MainScreen.png" width="40%" />
My take on a Flipper Zero type-aide.

## Usage
As usual **Back** closes the App.

**Splash Screen:**
- Icon and title at the top, scrollable text box underneath showing entered text or placeholder
- **Up**/**Down** arrows appear when you can scroll, **OK** opens keyboard to enter/edit text
  
**Keyboard screen:**
After entering text **OK** returns to the splash screen
